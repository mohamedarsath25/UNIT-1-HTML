**# UNIT-1-HTML**
This unit focuses on the fundamentals of HTML (HyperText Markup Language), which is used to create and structure web pages. It covers the basic HTML document structure, text formatting tags, lists, links, images, tables, and forms. It also includes semantic HTML elements that improve accessibility and organization of web content. Additionally, this unit demonstrates how to link multiple pages, embed external content like videos and maps, and design a simple multi-page website project.

**Topics Covered:**

HTML document structure

Headings, paragraphs, and formatting tags

Lists (ordered and unordered)

Links and navigation

Images, audio, and video

Tables and forms

Form validation

Semantic HTML tags

iframe and embedding

Multi-page website creation

**Outcome:**

By completing this unit, I learned how to create structured web pages and build a basic website using HTML.
